# SMS Spam Detection using Naive Bayes

This project detects whether an SMS message is spam or not using a Naive Bayes classifier.

## Tech Stack
- Python
- Scikit-learn
- Streamlit

## Algorithm
- Multinomial Naive Bayes
- TF-IDF Vectorization

## How to Run
1. Install dependencies  
   pip install -r requirements.txt

2. Train model  
   python model.py

3. Run app  
   streamlit run app.py
